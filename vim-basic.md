# VIM Basic

## Navigation

| key | function          |
| --- | ----------------- |
| `h` | move cursor left  |
| `j` | move cursor down  |
| `k` | move cursor up    |
| `l` | move cursor right |

## Changing Modes

| mode    | use-case                                                   |
| ------- | ---------------------------------------------------------- |
| command | default mode. Used for manipulating and viewing text files |
| insert  | types text                                                 |

| key   | function          |
| ----- | ----------------- |
| `esc` | exit current mode |
